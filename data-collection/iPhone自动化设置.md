## 📱 iPhone 端设置（3 分钟完成）

### 方法 1：使用切换控制（推荐，支持无限重复）

#### 步骤 1：启用切换控制

1. **设置 → 辅助功能 → 切换控制**
2. **打开"切换控制"开关**
3. **开关 → 添加新开关...**
4. **选择"屏幕"**
5. **点击"轻点"**

#### 步骤 2：创建手势

1. **返回切换控制主页面**
2. **点击"配方"（Recipes）**
3. **创建新配方...**
4. **名称**：自动浏览汉字
5. **分配开关**：屏幕
6. **自定手势...**

#### 步骤 3：录制手势

打开 CloudBrush，进入汉字浏览页面，然后：

1. **开始录制手势**
2. **执行以下动作**：
   - 点击屏幕中间（打开汉字详情）
   - 等待 1.5 秒
   - 点击左上角返回按钮
   - 等待 0.5 秒
   - 从右向左滑动（切换下一个字）
   - 等待 0.5 秒
3. **停止录制**
4. **保存手势**

#### 步骤 4：配置循环

1. **在配方中选择刚录制的手势**
2. **设置"重复次数"：9999**（或足够大的数字）
3. **保存配方**

#### 步骤 5：启动自动化

1. **打开 CloudBrush App**
2. **进入汉字浏览页面**（选择任意部首）
3. **激活切换控制**（三击 Home 键或侧边按钮）
4. **选择"自动浏览汉字"配方**
5. **放置 iPhone，让它自动运行**

**预计运行时间**: 约 2 小时完成 3500 个汉字

### 方法 2：使用辅助触控（简单，但可能需要手动重复）

#### 步骤 1：启用辅助触控

1. **设置 → 辅助功能 → 触控 → 辅助触控**
2. **打开"辅助触控"**

#### 步骤 2：录制手势

1. **自定顶层菜单...**
2. **点击一个图标 → 分配操作**
3. **选择"手势"**
4. **创建新手势...**
5. **录制**（同方法 1 的手势录制）
6. **保存**

#### 步骤 3：使用

1. **打开 CloudBrush**
2. **点击辅助触控按钮**
3. **选择刚录制的手势**
4. **手势会播放一次**
5. **需要手动重复点击播放**

**缺点**: 不支持自动重复，需要手动重复触发

---

### 方法 3：使用快捷指令（最简单，但自动化程度低）

#### 创建快捷指令

1. **打开"快捷指令" App**
2. **创建新快捷指令**
3. **添加动作**：
   - 打开 App: CloudBrush
   - 等待 2 秒
   - 运行 JavaScript（模拟点击）
4. **保存**

**缺点**: 快捷指令无法直接控制 App 内的 UI 元素
