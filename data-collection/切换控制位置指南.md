# 切换控制 - 详细位置指南

## 📍 完整路径

```
设置
  → 辅助功能
    → 切换控制
      → 配方
        → 创建新配方
          → 动作
            → 自定手势
              → 创建新手势 (录制界面)
```

---

## 🔍 每一步的详细说明

### 第 1 步：打开设置
- 点击主屏幕的 **设置** (齿轮图标)

### 第 2 步：找到辅助功能
- 在设置列表中向下滚动
- 找到 **辅助功能** (人形图标)
- 点击进入

### 第 3 步：找到切换控制
- 在辅助功能页面，找到 **互动** 分组
- 点击 **切换控制**

### 第 4 步：启用切换控制
- 打开顶部的 **切换控制** 开关
- 开关变成绿色 ✅

### 第 5 步：先添加开关（重要！）

**在继续之前，必须先添加开关**：

1. 在切换控制页面，点击 **开关**
2. 点击 **添加新开关...**
3. 选择 **屏幕**
4. 点击 **轻点**
5. 点击右上角 **存储**

现在你已经有一个"屏幕-轻点"开关了。

### 第 6 步：进入配方

1. 返回切换控制主页面（点击左上角"<切换控制"）
2. 找到 **配方** 选项
3. 点击 **配方**

### 第 7 步：创建新配方

1. 点击 **创建新配方...**
2. 会弹出输入框，输入名称：`自动浏览`
3. 点击 **好** 或 **存储**

### 第 8 步：分配开关

1. 在配方编辑页面，点击 **分配开关**
2. 选择 **屏幕 - 轻点**
3. 返回

### 第 9 步：添加动作

1. 在配方编辑页面，点击 **动作**
2. 向下滚动，找到 **自定手势**
3. 点击 **自定手势**

### 第 10 步：创建新手势

1. 点击 **创建新手势...**
2. 现在进入手势录制界面！

---

## 🎬 手势录制界面说明

进入录制界面后，你会看到：

- **顶部**：录制时间显示
- **中间**：黑色录制区域
- **底部**：录制/停止按钮

### 如何录制

**在开始录制之前**：
1. 退出设置
2. 打开 CloudBrush
3. 进入汉字浏览页面（选择一个部首）
4. 然后返回设置继续录制

**录制步骤**：

1. **点击"录制"按钮** (红色圆点)

2. **执行以下操作**：

   **操作 1**：点击屏幕中间
   - 在黑色区域的正中间点击一次
   - 会出现一个蓝色圆点标记

   **操作 2**：等待
   - 不要动，等待 1-2 秒

   **操作 3**：点击左上角
   - 在黑色区域的左上角点击一次
   - (模拟点击返回按钮)

   **操作 4**：等待
   - 再等待 0.5 秒

   **操作 5**：从右向左滑动
   - 从黑色区域右侧 → 滑向左侧
   - (模拟翻页)

   **操作 6**：等待
   - 等待 0.5 秒

3. **点击"停止"按钮**

4. **预览手势**：
   - 可以点击播放按钮查看录制的手势
   - 如果不满意，点击"重新录制"

5. **保存手势**：
   - 输入手势名称：`浏览汉字`
   - 点击 **存储**

---

## ⚙️ 设置循环次数

保存手势后：

1. 返回配方编辑页面
2. 找到你刚创建的手势
3. 点击手势右侧的 **(i)** 信息图标
4. 找到 **重复** 选项
5. 设置为 **自定**
6. 输入 **9999** (或 3500)
7. 点击 **存储**

---

## 🚀 启动自动化

设置完成后：

1. **打开 CloudBrush**
2. **进入汉字浏览页面**
3. **三击侧边按钮**（或 Home 键）激活切换控制
4. **点击屏幕上的蓝色浮动按钮**
5. **选择"配方"**
6. **选择"自动浏览"**
7. **点击"播放"** 或 **"开始"**

iPhone 会自动重复执行手势 9999 次！

---

## 📊 验证是否工作

启动后，观察：

1. iPhone 屏幕会自动点击和滑动
2. Mac 的监控脚本会显示 URL 数量增加
3. 每 50 个 URL 会自动下载

---

## 🛑 如何停止

如果需要停止：

1. **三击侧边按钮**（或 Home 键）
2. 点击浮动菜单
3. 选择 **停止**

---

## 💡 小贴士

1. **设置永不锁屏**：
   - 设置 → 显示与亮度 → 自动锁定 → 永不

2. **保持充电**：
   - 连接充电器，避免电量耗尽

3. **关闭通知**：
   - 设置 → 勿扰模式 → 开启
   - 避免通知打断自动化

---

## ❓ 找不到选项？

如果你在某一步找不到对应的选项，可能是因为：

1. **iOS 版本不同**：切换控制的界面可能略有不同
2. **需要先添加开关**：配方需要先有开关才能设置
3. **语言设置**：如果是英文系统，对应名称是：
   - 切换控制 = Switch Control
   - 配方 = Recipes
   - 自定手势 = Custom Gesture
